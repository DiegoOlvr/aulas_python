### Manipulando texto
*Ênfase* => para deixar o texto assim usamos ```*Ênfase*```
**Negrito** => para deixar o texto assim usamos ```**Negrito**```
_Itálico_ => para deixar o texto assim usamos ```_Itálico_```
~~Riscado~~ => para deixar o texto assim usamos ```~~Riscado~~```
